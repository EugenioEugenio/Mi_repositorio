 Игра Камень-Ножницы-Бумага


1.Описание
Проект позволяет играть с компьютером  в эту интереснейшую игру, но до счёта 10 с любой стороны, из-за опасений за психическое здоровье игрока. 


2.Как играть
Чтобы играть надо просто следовать подсказкам на экране, после каждого хода объявляется счёт, есть возможность прекратить игру до достижения счёта 10.


3. Функциональность
а. Функция досрочного выхода
 в. Функция счёта
 с. Функция останова при счёте 10.


4. Требования
Да никаких особенных требований, библиотеки random и time разве что.
